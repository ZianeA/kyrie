@NonNull fun fraction(fraction: Float): Keyframe<T>

Sets the time for this Keyframe, as a fraction of the overall animation duration.

Parameters

fraction - Float: The time associated with this Keyframe, as a fraction of the overall animation duration. This should be a value between 0 and 1.

Return
