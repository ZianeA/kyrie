ofPathMotion

@NonNull static fun ofPathMotion(path: Path!): Animation<PointF!, PointF!>

Constructs and returns an Animation that animates through PointF values in order to simulate motion along the given path. Clients can use to transform the returned animation into one that outputs floats corresponding to the path's x/y coordinates.

Parameters

path - Path!: The path to animate values along.

Return
Animation<PointF!, PointF!>: A new Animation.
